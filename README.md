mikinya-net
=========

https://mikinya.net/

- Host
  - [GitHub Pages](https://github.com/mikito/mikinya-net)
- Dependencies 
  - [Jekyll](https://jekyllrb-ja.github.io/)
  - [Minimal Mistakes](https://mmistakes.github.io/minimal-mistakes/)

## Memo

### Setup 

```sh
$ bundle install 
```

### Local Launch  

```sh
$ bundle exec jekyll s
```

### Deploy 

```sh
$ git push origin master 
```
